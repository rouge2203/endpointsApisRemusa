## 6. How to Filter by Car Type & Get ALL Compatible Parts

### Step-by-Step API Workflow

```
OBJECTIVE: Given a car → find ALL compatible autoparts

Step 1: Identify Vehicle Type
   API: GET /types/list-vehicles-type
   Returns: PC(1), CV(2), Motorcycle(3), LCV(4), etc.

Step 2: Find Your Vehicle
   Option A - By VIN:
     API: POST /vin/decoder (VIN_Decoder_v1/v2/v3 or all-in-one)
     Returns: vehicleId, modelId, manufacturerName, etc.

   Option B - By License Plate (Netherlands):
     API: Vehicles_by_License_Plates_of_the_Netherlands_Kenteken

   Option C - By KBA Number (Germany):
     API: Vehicles_by_KBA_number_Germany

   Option D - Browse by Manufacturer → Model → Vehicle:
     API: Models_by_Type__Manufacturer  →  Vehicle_IDs_by_Model_IDs

Step 3: Get All Part Categories for That Vehicle
   API: GET /category/type-id/{typeId}/products-groups-variant-3/{vehicleId}/lang-id/{langId}
   Tool: List_Categories_by_Vehicle_ID__v3

   Parameters:
     - typeId: 1 (PC), 2 (CV), 3 (Motorcycle), etc.
     - vehicleId: from Step 2
     - langId: 4 (English)

   Returns: Full category tree (Body, Engine, Steering, Brakes, etc.)

Step 4: Get All Parts for a Specific Category
   API: GET /articles/list/type-id/{typeId}/vehicle-id/{vehicleId}/category-id/{categoryId}/lang-id/{langId}
   Tool: Article_List_by_Vehicle_ID__Category_ID

   Parameters:
     - typeId: 1
     - vehicleId: from Step 2
     - categoryId: from Step 3 (e.g., 100198 for Steering > Joints)
     - langId: 4

   Returns: All compatible articles with supplier info & images

Step 5 (Optional): Get Detailed Specs for Any Article
   API: POST /articles/article-id-complete-details
   Tool: Article_Details__Compatibility_for_Article_ID

   Returns: Full specs, OEM cross-refs, compatible vehicles list
```

### Example: Get All Steering Joints for Subaru Forester SF 2.0 AWD

```
1. typeId = 1 (PC)
2. vehicleId = 8785 (Forester SF 2.0 AWD)
3. categoryId = 100198 (Steering > Joints)
4. Call: Article_List_by_Vehicle_ID__Category_ID
   → Returns 220 articles from all suppliers
```
